My Projects

Woodpecker - Simple core banking API


Developers can use it as a backend for a frontend app. They can users, back accounts, transactions (deposits and withdrawals). All APIs are authenticated using JSON Web tokens. Interest is calculated and deposited monthly using a cron style job.

Recognize - Document digitization and analytics


User provides a scanned document it is converted to text format and a simple editor is provided for formatting. It has a keyword based search which can search the contents of all the documents in the system. It also has an analytics suite build on top of the aforementioned feature that shows aggregate statistics for certain types of documents.

LetsKanban - A basic kanban board web UI


Users can manage user stories on a kanban board style UI with different colored cards for different priorities.They can post code snippets on the forum and others can comment. they can upload all relevant documents to a central repository.
