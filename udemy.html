Data Engineer's Path



"A scientist can discover a new star but he cannot make one. He would have to ask an engineer to do it for him."
- Gordon L Glegg



A Data Engineer is somewhat of a vague term. Data Science is all the hype nowadays and Software Engineer is always been a common term. Data engineers can be thought of as the 'Plumbers of Data Science' (Thanks to Andreas Kretz for this phrase). No matter how sophisticated a model is, It cannot add value if it does not make it into production. A data engineer's job is to make data pipelines in a data platform to ensure data is available to whomever needs it in a reliable and efficient manner.

Depending on the size of the company, the job ususally involves ETL development, scheduled workflow jobs or even Big data Processing. SQL is a must for any engineer. Whether you an modelling an operational system or building a data warehouse for reporting, SQL is king. Even in the big data world SQL is still of importance with Hive, Spark SQL & Apache kylin. Python is one the languages that is very popular in this field especially for startups or medium sized companies. If you are dealing with big data you might run into Java or Scala as well. Below are the courses I used to gain the skills for this job.

Learn SQL using PostgreSQL: From Zero to Hero

An all in one course for SQL Mastery. Everything from basics, Aggregation, Constraints, Joins, Subqueries, database design, Indexing to CTE, Window Functions, PL/SQL, Transactions, JSON Types & Triggers.



Data Warehouse Fundamentals for Beginners

An Intro to Data Warehousing. A very common part of a Data engineer's realm. Introduces the Core Concepts, Need for a Data Warehouse, Facts, Dimensional Modelling, Slowly Changing Dimensions etc.



Apache Spark for Java Developers

Complete Masterclass on Apache Spark. It includes all the core areas of Spark such as RDDs, Dataframes, Spark SQL, Spark ML, Spark Streaming and also includes Performance Optimization Tips, Integration with Kafka etc.



Apache Kafka: Real time Stream Processing

Get a complete understanding of Apache Kafka. Producers, Consumers, Brokers, Storage Model, Streaming via KTables, KStreams and even Testing Streaming applications.



Python for Data Science & Machine Learning Bootcamp

The ultimate Python course for Basics, Data manipulation, Data Visualization & Machine Learning. Covers Numpy, Pandas, Matplotlib, Seaborn, Scikit-learn and even Pyspark for Big Data Processing.
